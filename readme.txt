== Description ==
A Wordsearch, word find, word seek, word sleuth or mystery word puzzle is a word game that consists of the letters of words placed in a grid, which usually has a rectangular or square shape.

== Installation ==

= Minimum Requirements =

* PHP 8.0 or greater is required

= Installation =

Manual installation method requires downloading the Wordsearch plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
